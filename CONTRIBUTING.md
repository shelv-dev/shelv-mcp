# Contributing

Thanks for your interest in improving this package.

## Repository model

This repository is a public mirror of code developed in Shelv's private monorepo.

- We welcome issues and reproducible bug reports.
- We currently do not accept code pull requests for implementation changes.
- Maintainers sync approved changes from the private source repository.

If you open a pull request, maintainers may close it and port the proposed change internally.

## What to include in issues

- Package version
- Node.js version
- Minimal reproduction steps
- Expected vs actual behavior

## Release policy

Maintainers publish from this public repository. Releases follow semver.
